1. (1) 在 Node.js Command Prompt 或 
   (2) VSC 工作目錄 (終端機)
 執行指令：
 
 npm i node-fetch
 npm i node-html-parser
 npm i launch-browser
 
2. 查詢安裝過的 modules
 
 npm list --depth=0
 
3. 文件網址：
 
 node-fetch: https://www.npmjs.com/package/node-fetch
 launch-browser: https://www.npmjs.com/package/launch-browser
 node-html-parser: https://www.npmjs.com/package/node-html-parser
